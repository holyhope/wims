Les Fonctionalités de l'éditeur interactif (disponible dans le mode brut de Createxo) :

Menu horizontal :

Le menu horizontal contient des boutons de mise en page (HTML), d'autres qui permettent d'insérer les instructions Wims les plus fréquenteset enfin un bouton qui affiche l'aide que vous lisez.

Le premier groupe de boutons permet d'insérer des balises HTML de mise en page. Par exemple on sélectionne un mot, on clique sur le bouton I et le mot sélectionné sera entouré de la manière suivante :

<em>le mot sélectionné</em>.

Ce qui aura pour effet d'afficher le mot sélectionné en italiques.

Les autres boutons sont plus spécifiques au language OEF de WIMS, et sont regroupés en trois parties : avant, pendant, après :

  • Avant - Les instructions de cette partie s'exécutent avant le chargement de l'exercice. Ces instructions concernent généralement les métadonnées de l'exercice, comme son titre, le format d'affichage.
  • Pendant - La deuxième partie permet de gérer les affichages sur l'écran depuis le début de l'énoncé jusqu'à ce que l'utilisateur ait entré toutes ses réponses.
  • Après - La troisième partie permet d'analyser les réponses de l'utilisateur.

Liste des instructions

La liste des instructions contient l'intégralité des instructions du langage OEF.

Elle a trois fonctionnalités : insertion, documentation, auto complétion :

  • Insertion - On peut insérer des instructions de la liste en double cliquant ou via le bouton insérer la sélection.
  • Documentation - Avec un simple clic sur une instruction donnée, on accède aux informations concernant cette instruction : le groupe auquel elle appartient (latex ou wims), ses paramètres et un exemple d'utilisation éventuel.
  • Auto complétion - Les instructions wims et latex commencent par un antislash. Quand l'utilisateur commence à écrire une instruction, l'éditeur détecte cette action et propose un certain nombre de choix. Par exemple on tape \a et dans la liste des instructions s'affichent toutes les instructions qui commence par la lettre a. Avec un simple clic, on affiche l'aide de l'instruction selectionnée. En double cliquant, l'éditeur complète ce qu'on vient de taper.
