Interactive OEF editor Fonctionalities (used in Createxo raw mode) :

Horizontal menu :

Horizontal menu contains layering buttons (HTML), and buttons to insert some frequent WIMS instructions, and the last button displays this help page.

First button group let you insert HTML layering tags. First select a word in your code, then click on the I button, and your word will be embeded like this :

<em>selected word</em>.

This will present your word in an italic style in your exercice.

Other buttons are more specific to OEF language, and are grouped in 3 parts : "BEFORE", "DURING", and "AFTER" :

  • BEFORE - instructions in this part are executed before the exercice display. Most of them are exercice metadatas (title, author...)
  • DURING - This second part let you manage how your exercice will be displayed on the screen
  • AFTER - This last part let you analyse learner answers.

Instruction list

Here are listed all Wims instructions

You can interact with this list in 3 different ways :

  • Insertion - Just double-click on an instruction, or select one with a simple click, then click on the insert selection button below the list.
  • Documentation - With a simple click on an instruction, an associated help is displayed in the help box below the textarea.
  • Auto completion - When you type in an antislash (\) and some lowercase letters, the editor filters the instruction list to display only instructions starting with the letters you've typed.
